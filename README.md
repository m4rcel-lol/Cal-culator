# Cal-culator
Richard Watterson reviewed this game, but it recently had an update.

## New in this update
- Duo system: you now party up with **Richard Watterson (Pro Duo)** in battles.
- Richard can assist with bonus damage and a clutch heal when your HP is low.

## Requirements
- Python 3
- `pygame`
- `pypresence`

Install dependencies:

```bash
pip install pygame pypresence
```

## Run (Linux/macOS)
```bash
python3 calculator_rpg.py
```

## Run (Windows)
You can either run:

```bat
py -3 calculator_rpg.py
```

or use the included launcher:

```bat
py -3 run_windows.py
```
